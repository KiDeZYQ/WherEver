# Story 3.1: 命令词创建提醒

**Epic:** Epic 3 - 提醒创建与管理
**Story ID:** 3.1
**Status:** in-progress

## User Story

As a user,
I want to create reminders using voice commands,
So that I can quickly create reminders without typing.

## Acceptance Criteria

**Given** 用户点击麦克风按钮
**When** 用户说出提醒内容
**Then** 语音被识别为文字
**And** 显示识别结果供确认

**Given** 语音识别结果确认
**When** 用户确认
**Then** 创建提醒成功
**And** 显示成功提示

## Technical Notes

- Use speech_to_text package for voice recognition
- Support Chinese and English
- Parse time/location keywords from voice text
- Fallback to manual creation on failure

## Dependencies

- Story 2.1 (游客模式支持)
