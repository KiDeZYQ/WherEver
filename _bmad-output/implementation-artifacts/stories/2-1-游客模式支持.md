# Story 2.1: 游客模式支持

**Epic:** Epic 2 - 用户账户与订阅系统
**Story ID:** 2.1
**Status:** in-progress

## User Story

As a user,
I want to use the app as a guest without registration,
So that I can try the app functionality before committing to an account.

## Acceptance Criteria

**Given** 用户未登录
**When** 用户打开APP
**Then** 自动生成游客会话
**And** 游客可以创建最多3个提醒
**And** 游客数据存储在本地
**And** 游客可以升级为正式用户
**And** 游客会话过期时间为7天

## Technical Notes

- Guest session ID: UUID v4, stored in SharedPreferences
- Guest limit: 3 reminders (enforced locally)
- Session expiry: 7 days, checked on app start
- Upgrade flow: preserve guest data, link to new account
