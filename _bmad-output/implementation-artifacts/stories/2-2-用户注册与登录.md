# Story 2.2: 用户注册与登录

**Epic:** Epic 2 - 用户账户与订阅系统
**Story ID:** 2.2
**Status:** in-progress

## User Story

As a user,
I want to register and login with email/password,
So that I can securely access my account across devices.

## Acceptance Criteria

**Given** 用户访问注册页面
**When** 填写有效的邮箱、用户名和密码并提交
**Then** 账户创建成功
**And** 自动登录并跳转首页
**And** 显示注册成功提示

**Given** 用户访问登录页面
**When** 填写正确的邮箱和密码并提交
**Then** 登录成功
**And** JWT token存储到本地
**And** 跳转首页

**Given** 用户提交无效信息
**When** 表单验证失败
**Then** 显示相应的错误提示
**And** 不提交表单

## Technical Notes

- Validation: email format, password min 8 chars
- API endpoints: POST /api/v1/user/register, POST /api/v1/auth/login
- Token storage: FlutterSecureStorage
- Form state management: GetX (from Story 1.1 dependencies)
